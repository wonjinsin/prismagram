# prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow / Unfollow User
- [x] Edit my profile
- [ ] See user profile
- [ ] See my profile
- [ ] Log in
- [ ] See the full photo
- [ ] Upload photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
